# Stock Market Analysis Using Statistical Methods and Machine Learning

**Author:** Joseph Barragan  
**Course:** Python Programming | Kean University  
**Date:** June 2026

## Overview
Quantitative analysis of NVDA, AMZN, MSFT, and AAPL over 2020–2024 using statistical methods and machine learning.

## Methods
- Geometric Brownian Motion for price simulation
- Descriptive statistics, correlation analysis, rolling volatility
- One-sample t-test (hypothesis testing)
- Linear Regression (next-day return prediction)
- Random Forest Classifier (directional prediction)

## Requirements
pip install pandas numpy matplotlib seaborn scikit-learn scipy

## Usage
python stock_analysis.py

## Results
- NVDA achieved a Sharpe ratio of 2.01 and total return of 9061%
- Random Forest directional accuracy: 50.60% (consistent with EMH)
